## Neural Networks and Math with PyTorch

### Overview

This repository captures my hands on exploration of neural networks and PyTorch with a focus on understanding how the core pieces work together. I worked through each concept step by step and translated the underlying theory into working code

### Things I Learned

- Biological inspiration behind neural networks
- Perceptrons and their limitations
- Multi-Layer Perceptrons (MLPs)
- Forward propagation mathematics
- Common activation functions
- Loss functions (MSE, Cross-Entropy)
- Gradient Descent optimization
- Backpropagation algorithm
- PyTorch implementation of neural networks
